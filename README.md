# 📧 AI Email Generator

A simple Streamlit web application that generates professional emails using the **Groq API** and **LangChain Prompt Templates**.

---

## 🚀 Features

- Generate AI-powered emails
- Multiple email types:
  - Formal
  - Informal
  - Leave Application
  - Complaint
  - Appreciation
- User-friendly Streamlit interface
- Uses Prompt Templates for structured responses
- Powered by Groq's Llama 3 model

---

## 🛠️ Technologies Used

- Python
- Streamlit
- LangChain
- Groq API
- python-dotenv

## ⚙️ Installation

### Step 1: Clone the Repository

```bash
git clone https://github.com/your-username/LLM-Email-Generator.git
```

### Step 2: Open the Project

```bash
cd LLM-Email-Generator
```

### Step 3: Install Dependencies

```bash
pip install -r requirements.txt
```

### Step 4: Create a `.env` File

Add your Groq API key:

```env
GROQ_API_KEY=your_api_key_here
```

---

## ▶️ Run the Application

```bash
streamlit run app.py
```

The application will open automatically in your browser.
